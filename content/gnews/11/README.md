普京的質問似乎並非針對美軍。克里姆林宮發言人德米特裡·佩斯科夫10月31日說，俄羅斯政府掌握的情報顯示，一些非政府組織也在收集俄羅斯民眾的基因樣本，“總統自然知悉這些資訊”。

美方解釋 對照研究

同樣在10月31日，美國空軍教育訓練司令部發言人博·唐尼接受俄羅斯媒體採訪時澄清說，美國空軍“先進分子監測中心”必須收集俄羅斯人樣本的原因在於，研究人員已從一家美國公司處獲得另外一組來自俄羅斯的樣本。

俄方憂慮 為生物戰

儘管美方已做出解釋，但一些俄羅斯官員依然對包括美軍在內的外國團體採集俄羅斯人生物樣本有顧慮。

俄聯邦委員會（議會上院）國防安全委員會第一副主席弗朗茨·克林采維奇在社交媒體上寫道，“我不是說有人在準備針對俄羅斯開展生物戰”，但“一旦情勢突然需要”，並不排除出現這一場景的可能性。“不同族裔對生化武器反應不同也不是什麼秘密，這是要收集不同地區俄羅斯人生物材料的原因。”

曾在負責核查伊拉克和利比亞生化武器的聯合國特別委員會任職的伊戈爾·尼庫林說，一些人熱衷於收集俄羅斯人樣本也不是第一次了。這些活動都有個原則，就是必須要“斯拉夫裔歐洲人、主要是俄羅斯人”的樣本，“如果是外國組織，那麼他們如何處理樣本結果永遠無從所知”。

俄羅斯公民社會和人權理事會成員、俄紅十字會會長賴莎·盧庫措娃告訴俄羅斯衛星通訊社，為避免民眾的生物樣本被用於製造細菌武器，應監控此類收集活動。

據香港東網報導，俄羅斯總統普京本月初曾明言，未來屬於人工智慧（AI），誰在這技術上取得領先地位，誰就能主導世界。上週六（21日），他在俄羅斯索契出席世界青年節時，再就人類科技新突破提出警告。

普京指出，通過基因重組創造的“超人類戰士”不但將會成真，而且比核彈更可怕。

普京在索契青年節活動上提出警告

報導稱，普京在青年節上，談到基因重組科技時表示，在不久的將來，按照人類意願創造的人將會成真。

所創造的可以是數學家、音樂家，也可以是一個完全沒有恐懼、悲傷和痛感的軍人。

普京指出，這將帶來比原子彈更可怕的後果，他警告世界各國領袖，必須商議訂立法規，監管這種“超人類戰士”的創造。

普京強調，新技術應有效和廣泛地應用，但無論做什麼事，都有正反效果。人們要做的，應能為人類帶來好處，因此人類不應忘記道德倫理，因為缺少道德倫理，將導致人類滅亡。
