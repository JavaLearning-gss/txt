## 武漢封城時到底多少人死于CCP病毒？
`2020-04-26 07:13 GM30` [轉載自GNews](https://gnews.org/zh-hant/184949/)

**作者：Dr. X**

武漢封城時到底有多少人死于CCP病毒？這個答案大家都想知道。因爲制毒放毒的CCP沒有告訴全世界武漢真實的死亡數字，再加上世界衛生組織WHO和CCP的沆瀣一氣，全球各國輕視了CCP病毒，而導致了全球大流行。發稿時世界已有超過200多個國家和地區近3百萬人感染， 20多萬人死亡的慘況（根據目前這個數據算出的死亡率是7%）。這些數據並不完全正確，因爲很多國家現在並沒有能力進行大規模CCP病毒的檢測，同時有無症狀感染者的存在，最後因CCP病毒的特殊性，老年化特別嚴重的國家（比如意大利）感染和死亡率特別高（達13.5%）。 今天讓我們嘗試用CCP的專家們發表在世界頂級雜志上的報告來估算一下武漢封城時到底多少無辜的同胞死于邪惡的CCP病毒。

哈佛大學的頂級專家根據流行病的數學模型預測出全世界大約40-70%的成人會被傳染上CCP病毒。 請注意：這個40-70%是不包括感染兒童的數據。雖然我們從各國（包括中共國）的新聞報道和文獻裏都已看到兒童甚至剛出生的新生兒感染CCP病毒的諸多案例，且目前還沒有機構作出一個比較令人信服的兒童感染率，同時，中共學者2020年3月26日在《新英格蘭醫學雜志》（The New England Journal of Medicine 影響因子70.670）上發表的論文數據顯示，他們于2020年1月22號前，在武漢地區記錄的425例CCP病毒病例中，沒有15歲以下兒童感染的病例（武漢封城時間是2020年1月23日到2020年4月8日），所以這次的估算就不包括兒童了 （不過我的腦海裏還痛苦清晰的記著那個令人心碎的視頻畫面。相信很多朋友都在郭媒體上看過，那是一個中國的醫院，具體城市不詳，2個身穿防護服從頭到腳都很嚴實的工作人員把3個小孩子的屍體從擔架上裝進了一個屍袋。恐怕是死人太多，沒有足夠的屍袋）。

武漢兒童約占總人口比例的10%，下面的計算我們會去除兒童的人口占比。另外一點需要強調的是武漢政府2019年的資料顯示武漢的老齡化極其嚴重，60-100歲的老人約占到總人口 的47.16%！65-100歲的老人約占到總人口 的25.89 %。根據1956年聯合國《人口老齡化及其社會經濟後果》的劃分標准，當一個國家或地區65歲及以上老年人口數量占總人口比例超過7%時，就意味著這個國家或地區進入老齡化。意大利是全球老年人口比例最高的國家之一，65歲以上的老年人口比例高達35%。如果沒有這場CCP病毒的全球大流行，按照武漢政府2019年的人口普查結果，不過幾年，武漢在老齡化比例上就要大大超過意大利了。

首先，讓我們簡單回顧一下普通人感染了CCP病毒後會有怎樣的後果。 個人出現的臨床症狀會不相同：

無症狀（對本人沒事，可是對周圍其他人將會是個噩夢，因爲會傳染給別人），
發燒、全身乏力、幹咳、疼痛、和呼吸困難（需要或不需要住院治療，可能自愈也可能惡化成更嚴重的地步。有經曆者描述有瀕死感、會有全身被火車碾過一樣的疼痛，或是每次呼吸時感覺肺部疼的像是塞滿了玻璃渣子），
呼吸衰竭（病人沒法呼吸，或者就算能呼吸，肺部也無法産生足夠濃度的氧氣通過血液供給全身。需要重症監護病房ICU進行機械通氣和支持。簡單說就是需要呼吸機幫你呼吸，不管是呼吸機連著一根管子查到肺裏的還是呼吸機連著面罩的，又名無創呼吸機，英國首相鮑裏斯·約翰遜用的是後一種，其實病人體驗比前一種更難受），
敗血症（很危險，需要住進ICU治療），
敗血性休克（需要住進ICU治療，搶救不及時會很快死亡），
多器官功能障礙綜合征MODS或多器官衰竭MOF（能救活的可能性很小，即使救活了病人以後的生存和生活質量都不太樂觀）。
目前世界各地報道的案例中還有一些其他的症狀，比如說眼部感染、皮膚感染、腳部感染/畸形、嗅覺和味覺的喪失等等，只要不發展到需要住進ICU的地步，都只算輕度病症。

有一篇第一作者是中共專家的報告于2020年2月24日發表在《美國醫學會雜志》（JAMA， 影響因子51.275）上。此論文分享了中共國疾病預防控制中心的臨床和流行病學數據以及72314例病例的記錄（兒童病例報告的頻率低）。確診CCP病毒感染的病例占總人數的62％，其中1％的病人爲無症狀感染者。雖然這個數據不僅僅是在武漢收集的，但因爲樣本量巨大，所以非常有意義地佐證了哈佛專家的預測。

中共疾病預防控制中心根據CCP病毒臨床表現的嚴重程度將72314個病人分爲以下三類：

輕症：占所有案例的81％, 含非肺炎和輕度肺炎。
重症：占所有案例的14％，含呼吸困難，呼吸頻率≥30 / 分鍾（正常值是12-20分鍾），血氧飽和度（SpO2）≤93％（正常值是96-100%），PaO2 / FiO2 &lt;300 （正常值是400-500 mmHg），和/或肺部浸潤&gt; 50％（正常值是0，就是超過一半的肺裏都有液體，這樣的肺髒自然無法正常的工作）；
危重症：占所有案例的5％， 含呼吸衰竭，敗血性休克、多器官功能障礙（MODS）或多器官衰竭（MOF）。
中共疾病預防控制中心的報告稱CCP病毒的病死率爲2.3％（這個比例是全國的均值，不是特指武漢的）。報告還稱大約一半（49％）的危重患者（占所有感染病人的5%）會死亡（這個數據也比西方發達國家好，歐美發達國家的50-80%的危重患者會死亡），死亡病人都有心血管疾病，糖尿病，慢性呼吸道疾病和腫瘤病等既往合並症。死亡病例中約15％的是≥80歲老年患者，8％的是70-79歲的患者。

現在我們就用中共專家們在國際頂級雜志上發表的數據和哈佛專家的預測這兩種方法來算一算武漢封城期間到底死了多少人吧。武漢市長當時的電視講話中明確指出，除過年前離開武漢的5百萬人，武漢封城時還有9百萬人在城裏。中共疾病預防控制中心報告中說疾病確診率爲62％，其中1％病人的年齡在9歲以下，1％的病人年齡在10至19歲之間。那麽武漢就大約有5百58萬（9000000 X 62% =5580000）人會感染CCP病毒。在這些感染病人中的128340（5580000 X 2.3％死亡率）會死亡。如果我們用哈佛大學專家預測的40-70%的成人感染率來測算，減去人口中10%的兒童，武漢成人共有8百10萬。成人感染CCP病毒數會在3240000-5670000之間， 而成人死亡的人數會在74520 到130410之間。 所以按照2.3%的死亡率，武漢至少有大約12萬人（包括兒童）或者7萬到13萬（只是成人）在武漢封城期死于CCP病毒！更何況，介于武漢老齡化的程度和意大利接近，而意大利目前的死亡率是13.5%。如果我們用意大利的死亡率來算，武漢就會有40到70多萬成人死亡。我希望死亡率是2.3% ，因爲那樣的話我們就沒有那麽多同胞痛苦地死于CCP病毒。可是如果我們按照以下ICU病房的推理，就會發現這個幾十萬的數字恐怕會更接近真相。但無論是哪個數字，這都只是武漢一個城市的死亡數據。CCP竟然能厚顔無恥的宣稱全中共國才死了4632人，這還是追加了1千多人後的數字！

按中共疾病預防控制中心以上提及的81%感染CCP病毒的病人是輕症來算。輕症病人總人數在武漢是4百5十多萬人（9000000 X 62% X 81% = 4519800）！ CCP專家建議讓病人在家忍一忍，病毒就會給焖死了，可前提是誰能確定在家把CCP病毒焖死前沒有先被CCP病毒把自己毒死吧。根據武漢衛健委2020年2月11日的官方數據顯示武漢隨著CCP病毒確診數量越來越多，定點醫院和已投入使用的三所方艙醫院（4400床位）的床位已不夠用，加上被征用的十多所學校宿舍改造出的11455張床位。再加上陸續在建的另外12所方艙醫院（7200床位），總數一共有23055張可以收治輕症病人的床位。2萬3千張床對4百50萬病人，杯水車薪啊。

任何有醫學知識或常識的人都可以理解重症和危重症病人（根據中國疾病預防控制中心的報告，是19%）是一定需要住院治療，而且必須是ICU重症監護病房進行的治療，否則存活的機會基本爲零。另一篇中國專家們于2020年2月21日發表于《柳葉刀》 (The Lancet，影響因子59.102) 國際頂級醫院雜志上的論文顯示在2020年1月2日前武漢收治入院的41名CCP病毒患者有32%需要住進ICU進行救治（這個高ICU收治率完全可以理解，因爲第一波CCP病毒的毒性是最強、致死率是最高的）。即使我們采用保守的19%的ICU收治率，武漢當時需要住進ICU的重症和危重症病人至少1百萬人（9000000 X 62% X 19% = 1060200）。可疫情發生前的武漢總共有多少張ICU病床位呢？ 根據武漢市衛健委官網公布的數據顯示，武漢在2020年2月12日有35家定點醫院+ 雷神山和火神山醫院可以收治重症、危重和疑似危重病例，總共可以提供床位14408張（連一個零頭都不到）。就算住進ICU病人也不一定能活下來，更何況那些住不進去醫院的1百多萬危重症病人呢？不管他們是呆在家裏還是在方艙醫院，都是得不到所需的重症監護和治療的，能活下來絕對是不幸中的萬幸。這也可以解釋爲什麽武漢殡儀館的匿名工作人員說他們收到的非醫院死亡病人比醫院死亡的病人多的多。

同時還請注意，以上武漢的14408張病床只是可以收治危重症的醫院床位，並不是真正的ICU病床。根據《重症醫學：華東地區現狀調查(2015年第三次ICU普查)》顯示，2015年中國平均每10萬人擁有ICU床位數3.2張。在2019年時有中國專家樂觀認爲應該每10萬人能擁有5張ICU床位了。即便根據這個數據，按武漢有1千4百萬人口算，全市也只有700張ICU病床。臨時改成收危重症的病房根本不可能全部達到ICU病床對硬件和人員的要求。ICU病房不僅需要各種特殊配置、昂貴且操作複雜的醫療器械和呼吸機，還需要專業培訓過的ICU醫生和護士。這份報告同時也顯示，中共國不僅ICU床位占人口比低于西方發達國家， ICU醫護編制也不足。2015年中共國ICU醫師人數與床位數比約0.62：1，護士人數與床位數比約爲1.96：1（美國ICU護士人數與床位比達4：1），均低于《指南》的最低標准。就算CCP有錢又有人可以在9天內建好有2500張床的醫院，也完全沒有可能在9天裏裏培訓出2500個合格的ICU醫生、護士及熟練操作呼吸機的技師。

這就是爲什麽CCP要從全國各地派出總共32572名醫務人員去支援武漢了。加上原本武漢本身擁有的3.82萬醫務人員， 一共有7萬多人，看上去不少。可是人不能24小時不吃飯不睡覺的工作，而且因爲個人防護用品的極度缺乏，當時每個醫護人員每天做6或8小時輪班（這樣就能在6-8個小時內不喝水不吃飯不上廁所不離開工作的病房，如此就只需要一套防護用品，不會浪費了），這樣的話，7萬的人數就要除以3 或者4。而且理論上500或擁有更多床位醫院的醫院病床數和全職醫務人員配比率應該是1：10 。 武漢3萬7千多張非方艙床位（23055+14408）應該配置37萬全職的醫務人員才能保證他們能健康並精力充沛的給病人們提供最好的醫療服務。可當時武漢的床位和人員比只有1：2。這是遠遠不夠的。吃不好、休息不好、個人防護用品的缺乏、長時間的工作、CCP病毒AI病毒的邪惡特質、再加上沒有正確的信息，導致了很多可愛可敬的白衣天使們也染上CCP病毒、最後死在他們的工作崗位上，這是非常令人惋惜的。更讓人憤怒的是在抗擊疫情的時候，死一個警察CCP付10萬人民幣撫恤金，死一個醫務人員卻只有5000人民幣。這種做法不僅侮辱了白衣天使救死扶傷高貴品格和無私價值，也再一次證明了CCP不在乎老百姓的福祉，根本不關心老百姓的健康和死活 (從全球來說，所謂第二大經濟體的中共國，人口和醫務人員比例及全民健保都在最差的那個梯隊裏的。以後有空，我再和大家細聊 )。

根據世界重症與危重病醫學會聯盟數據，美國是每10萬人擁有的ICU床位數全球最高（34.7張）的國家，德國和加拿大每10萬人擁有的ICU床位數分別高達29.2 和13.5張。意大利、法國和韓國是12.5， 11.6， 和10.6張。這六個國家中美國的CCP病毒病死率現爲5.6%， 德國的病死率爲3.7%，加拿大是5.3%。意大利、法國（65歲以上的老年人口比例高達19.6%）和韓國的病死率是13.5%、14%和2.2%。 而既沒有足夠的ICU病房、沒有足夠的重症監護設備和器械、也沒有足夠的ICU醫護人員的中國的病死率只有2.3%，雖然中國有很多有知識有能力有情懷且工作努力的醫務人員，可惜巧婦難爲無米之炊，在沒有疫苗、沒有特效藥、沒有防護設備、只有讓你奉獻奉獻再奉獻的情況下，醫務人員連他們自己都保護不了，又如果去救治病人呢？ 完全解釋不通！在這次疫情上，CCP根本沒有能力救治感染病毒重症瀕死無助的老百姓，它的能力僅限于向世界瞞報和少報感染和死亡病例！當然CCP還有很多其它擅長的事，包括並不限于不學無術、自吹自擂、溜須拍馬、欺上瞞下、奴役壓榨、不自量力、昏庸無道、卻觊觎霸權。寫到這裏，我的總結就是郭先生在一次直播中悲憤的怒吼：假的！假的！全都是假的！
